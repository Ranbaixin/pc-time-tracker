## 1. 项目概览

**核心业务**：Windows 本地电脑使用时间追踪系统。追踪前台窗口、分类活动内容、生成统计报告、提供 REST API + MCP 工具 + HTML 仪表盘，支持接入 AI Agent 分析。

**技术栈**：
- Python 3.13 + FastAPI + uvicorn（REST API）
- SQLite（WAL 模式，线程安全，`data/tracker.db`）
- ctypes（Win32 API）+ psutil（进程、电池）
- MCP SDK（`mcp` + `httpx`，stdio JSON-RPC）
- Chart.js CDN（单文件仪表盘 `static/dashboard.html`）
- Pydantic v2（配置模型 + API 模型）
- Chrome/Edge 浏览器插件（`F:\browser-activity-tracker\browser-activity-tracker1.0`，MV3）

**端口**：默认 **8088**（8080 被其他程序占用）
**项目路径**：`C:\Users\Ran-xin\pc-time-tracker`
**GitHub**：`https://github.com/Ranbaixin/pc-time-tracker`

## 4. 关键架构决策

| 决策 | 原因 |
|------|------|
| **窗口切换按进程名比较**（不按标题）| 标题频繁变化导致碎片化。改成 `process_name != prev` 才切换，标题变只 update |
| **会话结束时间用最后活动时间**（不直接用 now）| 防止关机重启后出现 35 小时会话 |
| **时间线按实际时间比例分配**| 跨小时的活动不能全算在起始小时，最多 60 分钟/小时 |
| **开机自启用注册表 Run 键**| 比 VBS/启动文件夹更可靠。`HKCU\...\Run`，命令：`cmd /c cd /d DIR && pythonw -m src.main run` |
| **CORS 用 allow_origin_regex**| `r"https?://(localhost|127\.0\.0\.1)(:\d+)?$"` + 额外允许 `chrome-extension://*` |
| **MCP stdout 保护**| `logging→stderr` + `redirect_stdout` |
| **中文兼容：不用 `\b`**| 文件类型用 `endswith()` 和 `in`；站点匹配用 `in` + lowercase |
| **浏览器插件同步策略**| 不是每秒！事件驱动（tab 切换/窗口失焦）+ 30 秒定时批量。失败时累积不丢数据 |
| **分类引擎独立模块**| `ActivityClassifier` 单例，每次 classify < 0.5ms。用户自定义规则存 `app_categories` 表 |

```
GET       /api/v1/activity              # 活动记录（筛选+分页）
GET       /api/v1/activity/recent       # 最近 N 条

GET       /api/v1/stats/categories      # 分类占比
GET       /api/v1/stats/sites           # 站点排名
GET       /api/v1/stats/keywords        # 关键词频率
GET       /api/v1/stats/content_types   # 内容类型分布
GET       /api/v1/stats/file_types      # 文件类型分布

GET       /api/v1/agent/context         # Agent 综合上下文（含分类/站点/关键词）
GET       /api/v1/agent/raw             # 原始数据导出
GET       /api/v1/agent/insights        # 预计算洞察
GET       /api/v1/agent/prompt          # 预制 AI Prompt

GET/POST/DELETE /api/v1/classifications # 自定义分类规则
```

**所有返回格式**：`{"data": {...}}` 或 `{"data": {...}, "error": null}`

## 12. 快速故障排查

| 现象 | 可能原因 | 处理 |
|------|---------|------|
| `127.0.0.1:8088` 打不开 | 服务未运行 | `cd C:\Users\Ran-xin\pc-time-tracker && python -m src.main run` |
| 启动报 `ModuleNotFoundError: No module named 'src'` | 不在项目目录 | 必须 `cd` 到项目根目录再启动 |
| 注册表自启不生效 | 注册表键值可能被安全软件拦截 | 检查 `reg query HKCU\Software\Microsoft\Windows\CurrentVersion\Run /v PCTimeTracker` |
| 仪表盘空白 | 浏览器缓存旧 JS | `Ctrl+F5` 强制刷新 |
| 数据全是碎片 | 服务器还在跑旧代码 | 停掉重启 |
| API 返回 503 | uvicorn 未完全启动 | 等 3-5 秒重试 |
